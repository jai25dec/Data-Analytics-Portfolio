# Spotify Data Analysis

## Objective
Analyze Spotify music dataset to identify patterns in song features and popularity.

## Tools Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Correlation analysis of audio features
- Visualization of music trends

## Key Insights
- Danceability and energy influence song popularity
- Loudness has moderate correlation with popularity
- Tempo patterns vary across music genres
